# Minhas Tarefas

Minhas Tarefas é um projeto simples para treinar CRUD.
Nele o usuário cria suas tarefas, armazena no Local Storage e após realizadas o usuário as exclui.

## Tecnologias utilizadas:

- HTML
- CSS
- Javascript
